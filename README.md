# Поиск репозиториев React + TypeScript
![Скриншот поиска](screen.png)

## Технологии

* `React.js`
* `react-dom`
* `react-redux`
* `sass`
* `Material UI`

## Установка

Склонируйте репозиторий:

```bash
git clone https://github.com/yato90/github_search.git
cd yam-music
npm install
```
Укажите свой github token в .env для работы с GitHub GraphQL API

## Запуск

```bash
npm run dev
```
